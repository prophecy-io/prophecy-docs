---
title: Tableau
description: Properties required to create and update Tableau connections
---

Properties for Tableau connections are specified in the `properties` object when creating or updating a connection. All properties are nested within the connection request body.

<Tip>
  Create the secret containing your Tableau personal access token value in Prophecy before creating the Tableau connection. The secret must be of type `text` and contain the token value as its value.
</Tip>

## Properties

<ParamField body="serverUrl" type="string" required>
  Tableau server URL. Format: `https://{server}.online.tableau.com` for Tableau Online or `https://{server}/` for Tableau Server. Example: `https://10ay.online.tableau.com`.
</ParamField>

<ParamField body="siteName" type="string" required>
  Name of the Tableau site to connect to. This is the site identifier within your Tableau server or Tableau Online instance.
</ParamField>

<ParamField body="tokenName" type="string" required>
  Name of the personal access token used for authentication. This is the token name you specified when creating the personal access token in Tableau.
</ParamField>

<ParamField body="tokenValue" type="object" required>
  Reference to a Prophecy secret containing the Tableau personal access token value. The secret must be stored in Prophecy before creating the connection.

  <Expandable title="tokenValue properties">
    <ParamField body="tokenValue.kind" type="string" required>
      Secret kind. Must be `prophecy`.
    </ParamField>

    <ParamField body="tokenValue.subKind" type="string" required>
      Secret sub-type. Must be `text` for token values.
    </ParamField>

    <ParamField body="tokenValue.type" type="string" required>
      Type of reference. Must be `secret`.
    </ParamField>

    <ParamField body="tokenValue.properties" type="object" required>
      Properties identifying the secret by ID and name.
    </ParamField>

    <ParamField body="tokenValue.properties.id" type="string" required>
      The ID of the secret stored in Prophecy. You can find this in the secret details page.
    </ParamField>

    <ParamField body="tokenValue.properties.name" type="string" required>
      The name of the secret stored in Prophecy.
    </ParamField>

  </Expandable>
</ParamField>

<ParamField body="id" type="string">
  Optional connection ID. Can be an empty string if not provided.
</ParamField>
