---
title: 'Upload files'
description: 'Add local files to the SQL warehouse'
---

If you want the Agent to incorporate a certain file into your pipeline, you can upload it directly into the chat. Uploaded files are stored in the SQL warehouse configured in your fabric.

1. Click the paperclip icon in the chat footer.
1. Configure where the file should be stored.
1. Save the file.
1. The file will be converted to a table in the SQL warehouse.

The Agent will use the Table gem to read the file into your pipeline.
